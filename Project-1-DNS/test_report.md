# Test Report — DNS Project


## Общая информация

Проект: Тестирование интернет-магазина DNS  
Тип: Ручное тестирование


## Цель тестирования

Проверить основные функции сайта:
- поиск товаров
- карточка товара
- корзина
- фильтры


## Что было протестировано

- Поиск товаров
- Поиск с пустым значением
- Поиск с пробелами
- Поиск с цифрами
- Добавление товара в корзину
- Удаление товара из корзины
- Фильтр по цене


## Найденные дефекты

- Нет обработки пустого поиска
- Нет обработки пробелов в поиске

(Все дефекты носят некритичный характер)


## Итог

Сайт работает стабильно в основных сценариях пользователя.  
Критических ошибок не обнаружено.
