# Bug Reports — DNS Project


## BUG-001 — Нет реакции на пустой поиск

**Шаги:**
1. Открыть сайт
2. Нажать в поиск
3. Оставить поле пустым
4. Нажать Enter

**Фактический результат:**
Нет никакой реакции системы

**Ожидаемый результат:**
Система должна показать сообщение или обработать пустой ввод

**Серьёзность:** Minor  
**Приоритет:** Low

**Шаги:**
1. Открыть поиск

**Фактический результат:**
Нет реакции

**Ожидаемый результат:**
Система должна игнорировать пробелы или показать сообщение

**Серьёзность:** Minor  
**Приоритет:** Low
