# Пример бага

## Название

При вводе некорректного поискового запроса отображается пустая страница.

## Окружение

Google Chrome
Windows 11

## Шаги

1. Открыть сайт.
2. Ввести некорректный запрос.
3. Нажать Enter.

## Ожидаемый результат

Отображается сообщение "Ничего не найдено".

## Фактический результат

Отображается пустая страница.

## Severity

Minor

## Priority

Medium

## Статус

Open
