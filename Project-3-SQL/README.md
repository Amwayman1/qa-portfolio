# Project 3 — SQL

## Описание

Практика работы с SQL-запросами на уровне Junior QA.

В рамках проекта изучены основные запросы для получения и фильтрации данных из базы данных.

## Что входит в проект

- Простые SQL-запросы
- Фильтрация данных
- Сортировка результатов
- Подсчёт записей
- Примеры запросов для тестировщика

## Инструменты

- SQL
- PostgreSQL (или MySQL)
