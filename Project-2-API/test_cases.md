Тест-кейсы — API проект

TC-001 — Проверка GET-запроса

**Описание:**
Проверка получения данных по API.

**Метод:**
GET

**URL:**
https://jsonplaceholder.typicode.com/posts/1

**Шаги:**
1. Открыть Postman.
2. Выбрать метод GET.
3. Ввести URL.
4. Нажать Send.

**Ожидаемый результат:**
- Сервер возвращает статус 200.
- В ответе присутствуют поля userId, id, title и body.
- Значение id равно 1.

**Фактический результат:**
Все проверки выполнены успешно.

**Статус:**
Пройден.

## TC-003 — Создание новой записи (POST)

**Описание:**
Проверка создания новой записи через API.

**Метод:**
POST

**URL:**
https://jsonplaceholder.typicode.com/posts

**Шаги:**
1. Открыть Postman.
2. Выбрать метод POST.
3. Ввести URL.
4. Открыть вкладку Body.
5. Выбрать raw → JSON.
6. Вставить JSON с данными.
7. Нажать Send.

**Ожидаемый результат:**
- Сервер возвращает статус 201 Created.
- В ответе присутствуют поля title, body, userId и id.
- Поле id создано автоматически.

**Фактический результат:**
Сервер вернул статус 201 Created. В ответе присутствуют поля title, body, userId и id = 101.

**Статус:**
Пройден.
