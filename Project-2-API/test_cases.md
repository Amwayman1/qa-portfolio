Тест-кейсы — API проект

TC-001 — Проверка GET-запроса

**Описание:**
Проверка получения данных по API.

**Метод:**
GET

**URL:**
https://jsonplaceholder.typicode.com/posts/1

**Шаги:**
1. Открыть Postman.
2. Выбрать метод GET.
3. Ввести URL.
4. Нажать Send.

**Ожидаемый результат:**
- Сервер возвращает статус 200.
- В ответе присутствуют поля userId, id, title и body.
- Значение id равно 1.

**Фактический результат:**
Все проверки выполнены успешно.

**Статус:**
Пройден.
